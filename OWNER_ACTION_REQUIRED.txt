SECTION B — Payment Gateway (الأولوية الأولى بعد الـ Launch)
================================================================

الخيار الموصى به للسوق العربي: PAYMOB
  • موقع التسجيل: https://paymob.com/en/get-started
  • يدعم: بطاقات بنكية، فودافون كاش، انستاباي، Meza
  • Commission: ~2-3% لكل معاملة
  • Integration: بعد التسجيل ستحصل على:
      - PAYMOB_API_KEY
      - INTEGRATION_ID
      - IFRAME_ID

  بعد الحصول على الـ keys، أضفها في js/services.js:
  const PAYMOB_API_KEY = 'YOUR_KEY_HERE';
  const PAYMOB_INTEGRATION_ID = 'YOUR_ID_HERE';

الخيار الثاني (عالمي): STRIPE
  • موقع التسجيل: https://dashboard.stripe.com/register
  • يدعم: بطاقات Visa/MC الدولية
  • يحتاج: Backend server (Vercel Function أو Supabase Edge Function)
  • الأفضل لو بتستهدف السوق الأوروبي والأمريكي
